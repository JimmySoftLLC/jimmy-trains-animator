Instructions on how to add your own media:

Media goes in folders you create in the media folder in your home directory. Name these folders with logical names related to what is in them. For example Christmas music could go in a folder named Holiday.

Animation timing:

Time stamp mode allows you to change your animation timing. Once time stamp mode is turned on press the left button to start your animation. Once the media starts press the right button for each point that you want an animation timestamp. Time your button presses to key moments in the media. Continue to do this until the animation finishes. Then timestamp mode will be automatically turned off and your animation script will be ready to add animations to.

Setup light string:

Enter your light string in the order which each component is connected together by pressing neo-6, cane-4... When finished press "Save".

Test lights:

Valid volume levels are from 1 to 100

Animator web page name:

Use alphanumerics and dashes only. All names are automatically appended with .local:8083. So your website url will be something like "website-name.local:8083"

Animation scripts:

Animation scripts run the animation. To load the script press the button for the sound track you want to modify. Modify the script as needed and press save to save the script.

Animation script table:

LNNN_R_G_B = All lights NNN (0 All, 1 to 999) RGB 0 to 255
NMMM_I_XXX = Neo modules MMM (0 All, 1 to 999) I index (0 All, 1 to 6) XXX 0 to 255
BXXX = Brightness XXX 000 to 100
FXXX_TTT = Fade brightness in or out XXX 0 to 100, TTT time between transitions in decimal seconds
MAXXX/XXXX = Play Media, A (M play music, W play music wait, A play animation, P play list), XXX/XXX (folder/filename)
ZRAND = Random rainbow, fire, or color change
ZRTTT = Rainbow, TTT cycle speed in decimal seconds
ZFIRE = Fire
ZCOLCH = Color change
C_NN,..._TTT = Cycle, NN one or many commands separated by slashes, TTT interval in decimal seconds between commands
E = End script
