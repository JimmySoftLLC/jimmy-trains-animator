    Instructions on how to add your own media:

    Media goes in folders you create in the media folder in your home directory. Name these folders with logical names related to what is in them. For example Christmas music could go in a folder named Holiday.

    Animation timing:

    Time stamp mode allows you to change your animation timing. Once time stamp mode is turned on press the left button to start your animation. Once the media starts press the right button for each point that you want an animation timestamp. Time your button presses to key moments in the media. Continue to do this until the animation finishes. Then timestamp mode will be automatically turned off and your animation script will be ready to add animations to.

    Animation script table:

    LNZZZ_R_G_B = Neopixel lights/Neo 6 modules ZZZ (0 All, 1 to 999) RGB 0 to 255
    LXZZZ_R_G_B = Lifx lights ZZZ (0 All, 1 to 999) RGB 0 to 255
    LPZZZ_YYY = Lifx lights power ZZZ (0 All, 1 to 999) YYY ON or OFF
    NMZZZ_I_XXX = Neo 6 modules only ZZZ (0 All, 1 to 999) I index (0 All, 1 to 6) XXX 0 to 255
    BNYYY = Brightness (Neopixel lights/Neo 6 modules) YYY 000 to 100
    FXXX_TTT = Fade brightness (Neopixel lights/Neo 6 modules) in or out XXX 0 to 100, TTT time between transitions in decimal seconds
    IXXX/XXX XXX/XXXX(folder/filename)
    MALXXX = Play file, A (P play media, W play media wait, S stop media), L = file location (M movies, E elves, B bells, H horns, T stops, S santa, Z christmas story, C cut, R recording, X shutter, Q quotes) XXX (file name, if RAND random selection of folder, SEQN play next in sequence, SEQF play first in sequence)
    MBRXXX = Music background, R repeat (0 no, 1 yes), XXX (file name) must be in first row all others ignored
    ZRAND = Random rainbow, fire, or color change
    ZRTTT = Rainbow, TTT cycle speed in decimal seconds
    ZFIRE = Fire
    ZCOLCH = Color change
    ZL_S_E_T_I = Scene change S start E end using (daylight,afternoon,....), time, increments
    ZN_S_E_T_I = Scene change S start E end using (red,green,....), time, increments
    ZJ = Joke
    SNXXX = Servo N (0 All, 1-4) XXX 0 to 180
    QXXX = Add command to queue (for example QAN_YYY is an Animation with YYY filename)
    API_UUU_EEE_DDD = Api POST call UUU base url, EEE endpoint, DDD data object i.e. {"an": data_object}
    WOPEN_xxxx = Open webpage, XXX web url
    WCLOSE = Close webpage, XXX web url
    E = End script
